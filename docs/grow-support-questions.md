# הודעה מוכנה לתמיכת Grow (העתק-הדבק)

לשלוח לתמיכה/אינטגרציה של Grow (משולם) כמו שהיא — במייל או בצ'אט. בלי הפרטים
שבסעיף 2 אי-אפשר להתחיל בדיקות Sandbox (הקוד עצמו כבר מוכן, ענף `grow`).

---

שלום רב,

אנחנו "לב התחביב" — חנות ציוד אמנות ויצירה ברחובות. האתר שלנו הוא אתר סטטי עם
Cloudflare Worker בצד השרת, ואנחנו בשלבי חיבור הסליקה מול Grow דרך ה-Light API.
נשמח לעזרתכם בכמה נקודות:

1. נבקש לאשר שמסלול המבצע שסוכם (₪59 לחודש + 0.6% עמלה) מחובר לחשבון שלנו.

2. נבקש פרטי Sandbox ל-Light API: ‏userId, ‏pageCode של Growin Wallet, וגם pageCode
   רגיל (redirect) כגיבוי. כתובת האתר: https://lev-hatahbiv.nivsasi.workers.dev
   (בהמשך יחובר דומיין קבוע).

3. האם מסלול המבצע דורש מסלול חשבון מסוים (Basic / Extra / Plus)?

4. מה העלות של 3DS בסליקה באתר, והאם אתם ממליצים להפעיל אותו בחנות אונליין?

5. נבקש לוודא שהפקת חשבוניות אוטומטית פעילה בחשבון, ומה נדרש מצידנו כדי לקבל את
   ה-invoice callback (אנחנו מעבירים invoiceNotifyUrl בקריאת createPaymentProcess).
